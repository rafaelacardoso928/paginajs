Exercícios JavaScript

Escolha um exercício para começar

1. Par ou Ímpar

Descubra se um número é par ou ímpar.

2. Contador com Passo

Crie uma contagem de números com intervalo.

3. Calculadora de IMC

Calcule o Índice de Massa Corporal (IMC).

04. Simulador de Notas

Simule a média e status das notas escolares.

05. Verificador de Idade

Calcule a idade da pessoa e informe se ela é menor, adulta ou idosa.

06. Gerador de Tabuada

Mostre a tabuada de 1 até 10 com o número escolhido.

07. Lista de Compras

O usuário digita itens e os adiciona a uma lista dinâmica.

08. Verificador de Palíndromos

O sistema verifica se é um palíndromo.

09. Simulador de Semáforo

Semáforo visual com botões para cada cor

10. Conversor de Temperatura

Converta uma temperatura de Celsius para Fahrenheit e Kelvin.

11. Simulador de Caixa Eletrônico

O usuário digita um valor e o sistema mostra quantas notas de cada valor serão usadas.

12. Cronômetro

Cronômetro funcional com botões de iniciar, pausar e zerar.

13. Gerador de Senhas Aleatórias

O usuário escolhe o tamanho da senha e clica em "Gerar". A senha gerada deve conter letras, números e símbolos.

14. Simulador de Sorteio

Receba dois valores e sorteie um número entre eles.

15. Enquete com Votos

Crie uma enquete com pelo menos 3 opções. Exiba o total de votos e as porcentagens para cada opção..
